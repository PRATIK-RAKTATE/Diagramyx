# User Story: US-013

**Title:** User SignUp/Register

**Module:** Authentication

**Actor:** Guest User

## Epic

User Authentication

## Persona

Individual User

## User Story

As a guest user

I want to sign up using my email, password, OAuth

So that i can access the login page with option to enter email and password and login button.

So that I can access the platform features securely.

## Acceptance Criteria

- All credentials must ne entered.

- User data saved in database.

## Security Considerations
- Threat risks: brute force, credential stuffing
- Mitigation: rate limiting, logging, monitoring

## Performance Expectations
- Login response < 200ms

## Traceability
- Workflow: Daily Usage
