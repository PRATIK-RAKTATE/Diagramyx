# Attach-to-Shape System

## Epic

Object Anchoring & Constraints

## Persona

Designer / Diagram User

## User Story

As a user,
I want objects to attach to shapes,
So that related elements stay connected when I move or edit the canvas.

## Acceptance Criteria

- User can attach objects to a parent shape.
- Attached objects move automatically with the parent.
- Attachment points snap to valid anchor positions.
- User can detach objects at any time.
- Visual feedback shows attachment state.
- Nested attachments are supported.

## Performance Expectations

- Attachment updates occur instantly during drag.
- No visible lag with multiple attached objects.
