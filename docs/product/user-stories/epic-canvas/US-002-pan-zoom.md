# Pan & Zoom Actions

## Epic

Drawing Workspace

## Persona

Designer/Visual Thinker

## User Story

As a user,
I want smooth zoom and pan controls,
So that I can move and navigate in large canvas easily.

## Acceptance Criteria

- User can pan canvas using mouse drag.
- User can zoom in and out in canvas using mouse scroll.
- Zoom centers around mouse cursor or focal point.
- minimum and maximum zoom in-out limits enforced.
- Pan and zoom do not affect the objects.

## Performance Expectations

- Pan response latency under 16 ms.
- Zoom animation remains smooth at 60 FPS.
- No visible stutter with typical canvas load.
