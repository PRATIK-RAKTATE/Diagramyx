# Drag and Resize Controls

## Epic

Object Manipulation System

## Persona

Designer

## User Story

As a user,
I want to drag and resize the objects,
So that I can move, delete, or resize the objects collectively or individually.

## Acceptance Criteria

- User can drag and resize the objects accurately.
- User can move or delete the number of objects at same time.
- User can update the same type of objects in one click.
- minimum and maximum limits are enforced

## Performance Expectations

- Drag operations feel instant with no input lag.
- Resize interaction updates at 60 FPS.
- No stutter with multiple selected objects.
