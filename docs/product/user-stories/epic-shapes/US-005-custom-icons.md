# Custom Icons

## Epic

Asset & Icon Management

## Persona

Designer

## User Story

As a user,
I want to upload and use custom icons,
So that I can personalize my canvas with brand or project-specific visuals.

## Acceptance Criteria

- User can import the custom icon according to his needs.
- The imported icons are visible in assets.
- User can drag the icons in canvas.
- Icons can be resized without change in quality.
- The icons remain on canvas after session reload.
- User can delete or reupload the icons

## Performance Expectations

- Icon upload completes within 2 seconds for typical file sizes.
- Dragging icons onto canvas feels instant.
- Rendering icons does not degrade canvas performance.
