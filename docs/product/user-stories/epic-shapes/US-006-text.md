# Text

## Epic

Canvas Annotation System

## Persona

Designer

## User Story

As a user,
I want to add and edit the text in canvas,
So that I can label and explain the ideas in the canvas clearly.

## Acceptance Criteria

- User can add,edit and delete the text in the canvas.
- User can change the text font, style, and color.
- Text can be resized and moved.
- Text remains readable in all zoom levels.

## Performance Expectations

- Text editing updates instantly without lag.
- Rendering remains smooth with many text objects.
- Cursor and selection remain responsive during typing.
