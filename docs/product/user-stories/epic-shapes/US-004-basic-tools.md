# Basic Design tools

## Epic

Drawing toolkit

## Persona

Designer/Visual Thinker

## User Story

As a user,
I want basic design tools like shapes, lines, etc.,
So that I can have basic tools to accurately design my idea.

## Acceptance Criteria

- User can use basic shapes like circle, square, and more.
- User can fill the shapes with color and change colors of borders as needed.
- User can duplicate the object.
- User can easily access the tools from the toolbar.

## Performance Expectations

- Shape rendering occurs instantly after creation.
- Editing actions show no visible delay.
- Canvas remains responsive with multiple active objects.
